# 🚀 프로젝트 제안: 깃허브 계정 스위처 (GitHub Account Manager)

**사용자가 여러 개의 깃허브 계정을 사용하면서 겪는 인증 충돌(403 Error)과 토큰 관리의 번거로움을 해결하기 위한 '코드 복사 & 설정 가이드' 미니 웹앱 기획안입니다.**

---

## 1. 프로젝트 배경 및 목적
- **페인 포인트(Pain Point)**: 한 컴퓨터에서 여러 개발 계정(개인, 업무 등)을 쓰다 보면 시스템 자격 증명 충돌로 인해 푸시가 거부되는 경우가 많음.
- **해결 방안**: 복잡한 시스템 설정을 건드리지 않고, 현재 프로젝트의 원격 주소에 토큰을 직접 적용하는 명령어를 자동 생성해줌으로써 누구나 쉽게 계정을 전환할 수 있도록 도움.

## 2. 주요 기능 (Core Features)

### 🧩 토큰 기반 리모트 명령어 생성기
- **입력**: 깃허브 저장소 주소(HTTPS), 발급받은 PAT(토큰).
- **출력**: `git remote set-url origin https://<TOKEN>@github.com/...` 명령어 자동 완성 및 클릭 한 번으로 복사.

### 👤 프로필 저장 기능 (Local Storage)
- 자주 사용하는 깃허브 계정명과 이메일을 '프로필'로 저장.
- 계정 전환 시마다 `git config user.name/email` 명령어를 함께 생성하여 '잔디(Contribution)' 유실 방지.

### 📖 토큰 발급 가이드 (Step-by-Step)
- 깃허브 설정 페이지에서 어떤 권한(Scope)을 체크해야 하는지 이미지와 텍스트로 친절하게 안내.

## 3. 기술 스택 제안 (Tech Stack)
- **Framework**: Next.js (App Router) 또는 Vite + React
- **Styling**: Tailwind CSS (웜톤 학교 테마 또는 깔끔한 다크모드 대시보드)
- **Security**: **중요!** 모든 데이터는 브라우저 내부 `LocalStorage`에만 저장하며 외부 서버로 절대 전송하지 않음 (100% 보안 강조).
